1. Handedness Distribution

At the whole corpus level, we find an approximate 55/45 percentage split in right-handed versus left-handed sums for all four Ascher pendant sum relationships:

  • pendant-pendant sums (54/46)
  • pendant sums by color (56/44)
  • pendant sums by index (57/43)
  • subsidiary pendant sums (57/43)”

2. Khipus with Left/Right Ascher Sums (Either vs Both)

In most cases (63% across the four named relationships), any given specimen presents both left- and right-handed sums

Code 
def show_either_both_sums(df, df_name):
    has_either_mask = [(a > 0) or (b > 0) for (a,b) in zip(df['num_left_sums'].values.tolist(), df['num_right_sums'].values.tolist())]
    has_either_sum_df = df[has_either_mask]
    has_both_mask = [(a > 0) and (b > 0) for (a,b) in zip(df['num_left_sums'].values.tolist(), df['num_right_sums'].values.tolist())]
    has_both_sum_df = df[has_both_mask]
    print(f"{df_name} num either_sums = {len(has_either_sum_df)} num_both_sums = {len(has_both_sum_df)}")
    return (len(has_either_sum_df), len(has_both_sum_df))

pps_df = pd.read_csv("../data/ascher_sum_relations/pendant_pendant_sum.csv")
ppc_df = pd.read_csv("../data/ascher_sum_relations/pendant_pendants_color_sum.csv")
ips_df = pd.read_csv("../data/ascher_sum_relations/indexed_pendant_sum.csv")
sps_df = pd.read_csv("../data/ascher_sum_relations/subsidiary_pendant_sum.csv")

either_pps, both_pps = show_either_both_sums(pps_df, "pps_df")
either_pps_color, both_pps_color = show_either_both_sums(ppc_df, "ppc_df")
either_indexed_pps, both_indexed_pps = show_either_both_sums(ips_df, "ips_df")
either_sub_pps, both_sub_pps = show_either_both_sums(sps_df, "sps_df")

sum_eithers = either_pps + either_pps_color + either_indexed_pps + either_sub_pps
sum_both = both_pps + both_pps_color + both_indexed_pps + both_sub_pps
print(f"sum_eithers = {sum_eithers} sum_both = {sum_both} ratio={round(100.0*float(sum_both)/float(sum_eithers))}%")
